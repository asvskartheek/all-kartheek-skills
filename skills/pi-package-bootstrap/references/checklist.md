# Pi Package Install Checklist

- [ ] Pi docs reviewed first
- [ ] target repo inspected for `package.json` + `pi` manifest or convention dirs
- [ ] install source chosen (npm/git/local)
- [ ] global vs project scope chosen correctly
- [ ] settings/package list verified after install
- [ ] resource counts checked (skills/extensions/prompts/themes)
- [ ] `/reload` or restart requirement communicated
